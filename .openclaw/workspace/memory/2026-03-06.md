## Saturation Model Flaw and Fix

During P1 investigation, discovered flaw in saturation model:
- Original: `P(t) = baseline + (1 - baseline) * (1 - exp(-t/sat_steps))`
- Problem: Causes non-monotonic behavior (peak then decay)
- Corrected: `P(t) = baseline + (1 - baseline) * exp(-t/sat_steps)`
- Validates: `P(0)=1`, `P(∞)=baseline`, strictly monotonic decay

Saturation demo (`saturation_demo.py`) created to illustrate finite-size plateau concept.
