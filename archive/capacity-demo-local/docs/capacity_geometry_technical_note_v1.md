# Capacity Geometry Technical Note (v1)

**Artifact**  
`outputs/figures/capacity_visual_proof.png` — generated by `scripts/make_capacity_visual.py` in this repo.  
Inputs: deterministic substrates, tuned PowerLawFilter parameters per substrate, 25-point capacity sweep, plateau window matching Option B validation.

**Substrates & Filters**  
1. Periodic lattice (D=2, N=32).  
   - Filter: PowerLawFilter(d0=2).  
   - σ grid: geomspace(0.5, 300, 300); plateau window [5, min(0.4·N²/(4π²), 100)].  
2. Random geometric graph (n=200, D=2, radius=0.35, seed=42).  
   - Filter: PowerLawFilter with β_max=max(d0, 1.5), γ=0.85, λ₀ quantile=0.6.  
   - σ grid scaled by median eigenvalue; plateau window [0.02/λ_med, 5/λ_med].  
3. Small-world graph (n=200, k=6, p=0.3, seed=42).  
   - Filter: PowerLawFilter β_max=2.5, γ=0.7, λ₀ quantile=0.7 with cached λ₀.  
   - σ grid scaled by median eigenvalue as above.  

**Computation**  
For each C in linspace(0.05, 1.0, 25):
```
weights = filter.apply(eigenvalues, C)
ds(σ), lnP(σ) = filtered_spectral_dimension(eigenvalues, weights, σ-grid)
plateau stats = extract_plateau(ds, σ-grid, σ_lo, σ_hi)
```
We store ds_plateau(C) for the curve and capture lnP vs lnσ for the first and last C values.  

**Figure Layout**  
- Left column: ds_plateau vs C (orange).  
- Right column: ln P vs ln σ overlay for C_min (blue) and C_max (green).  
- Grid of three rows = {lattice, RGG, small-world}.  

**Observations**  
- Lattice revisits the canonical staircase: ds ≈ 1.9 → 2.04.  
- RGG shows ds ≈ 2.4 → 2.81 with strong ln P slope change, proving spectral filtering replicates the capacity staircase without separability.  
- Small-world ds ≈ 0.74 → 0.97 while ln P slope steepens; random rewiring no longer hides the invariant.
