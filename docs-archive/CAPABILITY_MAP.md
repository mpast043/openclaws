# CAPABILITY_MAP.md

This is a quick mapping from common asks to OpenClaw tool groups and/or skills.

1. Code changes, tests, repo edits
Tools: group:fs, group:runtime, group:sessions, group:memory

2. Web research and source checks
Tools: group:web, group:ui (optional browser control)

3. Automations, reminders, scheduled runs
Tools: group:automation (cron, gateway)

4. Messaging integrations (Slack/Discord/etc.)
Tools: group:messaging (plus any channel plugins)

5. Multi-machine or macOS execution
Tools: group:nodes + Exec approvals configuration
